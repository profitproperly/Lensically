## The 20-part implementation scope

1. Autonomous cycle receipt
   - One immutable run record with trigger, cycle ID, start/end, observed state, occupied/missing slots, all consulted data, diagnosis, thesis, horizon plan, candidates, internal rejections, source/hypothesis, model evaluations, gates, coverage checks, scheduled IDs, unresolved issues, and completion evidence.

2. Strategy versioning
   - Durable strategy versions with beliefs, family states, evidence, confidence, changes, reversal conditions, experiments, and next tests.

3. Post hypothesis records
   - Before publication, record expected audience reward, response type, hook/premise rationale, exploitation/exploration status, comparables, expected range, and uncertainty.

4. Performance evaluation at maturity
   - Evaluate posts at 6/12/18/24 hours using observable engagement, rates, velocity, and distribution state. Exclude follower attribution.

5. Comparable-post evaluation
   - Compare by age, family, hook, structure, reward, format, time range, and maturity instead of comparing every post indiscriminately.

6. Multi-level learning
   - Learn separately at exact-post, hook, premise, payoff, emotional reward, structure, topic, family, source, generation mode, placement, and sequence levels.

7. Semantic repetition detection
   - Detect repeated underlying questions, tensions, financial scenarios, rewards, sentence architectures, openings, closings, and meanings even when wording differs.

8. Exposure ledger
   - Structured recent/future inventory of families, hooks, premises, payoffs, dollar amounts, questions, scheduled similarities, and repetition density.

9. Adaptive portfolio allocation
   - Evidence-driven family states: franchise, core, emerging, prospect, cooling, dormant. No permanent fixed quotas.

10. Controlled exploration system
    - Experiments with IDs, hypotheses, comparison groups, maturity windows, result criteria, and follow-up decisions.

11. Saved Pattern intelligence
    - Enrich patterns with verified metrics, stable identity, hook, mechanism, reward, structure, topic, tension, adaptation options, similarity, prior uses, results, confidence, and reuse state.

12. Source-to-results lineage
    - Complete chain: Saved Pattern or hypothesis -> source card/family -> generation decision -> scheduled post -> published post -> metric checkpoints -> evaluation -> strategy lesson.

13. Automatic learning brief
    - Mature-post synthesis covering improvements, weakening, uncertainty, family opportunities, fatigue, disproven assumptions, and next-run tests.

14. Confidence-based strategy changes
    - Every belief carries sample size, evidence window, supporting/contradicting evidence, confidence, maturity, and state transitions.

15. Operator benchmarks
    - Coverage accuracy, duplicate rate, semantic repetition, premise diversity, generic-content rate, gate failures, lineage completeness, prediction accuracy, strategy calibration, engagement-floor movement, median performance, high-performance hit rate, new-family discovery, winner preservation, and experiment learning rate.

16. Run-to-run self-comparison
    - Compare each cycle with prior cycles for diversity, hypothesis quality, generic output, prediction calibration, evidence use, repeated mistakes, and actual strategy influence.

17. Conversational audit layer
    - Read-only audit surfaces for conversational ChatGPT and owner to inspect evidence, conclusions, strategy changes, benchmark movement, missing capabilities, and engineering opportunities.

18. Noninterference operating rule
    - Persist the owner’s noninterference policy while preserving account safety, protected boundaries, and optional system-level audits.

19. Account-level follower checkpoint
    - Current total, distance to one million, and long-horizon trajectory only. No inferred attribution to posts, days, families, experiments, or posting periods.

20. Intelligence dashboard
    - Show strategy version, beliefs/confidence, family states, experiments, learning brief, benchmark history, engagement-floor trajectory, prediction accuracy, repetition trends, run receipts, lineage, and follower checkpoint.
