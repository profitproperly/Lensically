# Automation Checklist

Use automation to remove repeated work. Do not automate a broken offer.

## Minimum Setup

- One product file
- One landing page
- One payment link
- One email capture form
- One delivery email
- One content calendar
- One tracking sheet

## Daily Automation

- Pull yesterday performance
- Identify posts with response
- Identify posts with no response
- Update strategy memory
- Generate tomorrow content
- Check duplicate posts
- Schedule posts

## Weekly Review

- Which topic got the most response?
- Which post format got ignored?
- What question did people ask?
- What objection appeared?
- What product section needs improvement?
- What should be repeated next week?

## Guardrails

Never automate fake claims.

Never automate spam replies.

Never publish without duplicate checks.

Never ignore performance data.
