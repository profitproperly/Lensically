# Manifest Mental Slate Builder

## Overview

The slate is 17 attempts to win, not a content calendar of categories. Every post should be selected because it has a plausible path to outperform the current floor or attack the top-post target.

## When To Use

Use this skill when:

- Generating 17 posts for 7am-11pm.
- Choosing between candidate directions.
- Balancing exploitation of winners with audience fatigue.

## Procedure

1. Use winner analysis first.
2. Define what today must beat: top likes, top views, average likes, average views.
3. Generate more internal candidates than needed.
4. Cut anything bland, abstract, or category-filling.
5. Keep the 17 strongest candidates, not the 17 most diverse.
6. Run fatigue review.
7. Return only slot/text unless explicitly asked for rationale.

## Slot Logic

Slots are distribution containers, not creative categories. Do not assume:

- morning equals money
- lunch equals spiritual
- evening equals love
- night equals blessing

If timing matters, it should be because the sentence fits likely reader state, not because of a hard-coded genre plan.

## Quality Bar

A candidate must answer:

- What is the reward?
- Why would this beat the current average?
- Is the picture clear enough to stop the scroll?
- Is the ending stronger than the setup?
- Is it different from recent winners in its payoff?

## Verification Checklist

- [ ] The slate uses current performance data.
- [ ] It attacks the 2x top-post target and rising average floor.
- [ ] It does not fill fake categories.
- [ ] It returns only the requested output fields.
